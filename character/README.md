# Character Queries

This folder contains GraphQL queries to fetch character details from the Rick and Morty API.

Each query uses:

character(id: ID!) {
  id
  name
  status
  species
  type
  gender
}

Files included:
- character-id-1.graphql + output
- character-id-2.graphql + output
- character-id-3.graphql + output
- character-id-4.graphql + output
